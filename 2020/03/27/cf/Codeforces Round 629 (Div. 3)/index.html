A Divisibility Problem

题目描述

给定两个正整数 aaa 和 bbb，你可以在一步操作中将 aaa 加上 111。你需要找到最少需要多少步操作可以使得 $a \mid b $。存在最开始就满足 a∣ba \mid ba∣b 的可能从而只需要 000 步操作。

输入格式

第一行一个正整数 t(1≤t≤104)t( 1 \le t \le 10^4)t(1≤t≤104)。

之后 ttt 行每行两个正整数 a,b(1≤a,b≤109)a,b(1\le a,b \le 10^9)a,b(1≤a,b≤109)。

输出格式

输出共 ttt 行，每行一个整数，表示能达到 a∣ba\mid ba∣b 的最小操作步数。

解法

题面上描述的是 a∣ba \mid ba∣b 也就是 aaa 能够整除 bbb ，对于 a∤ba \nmid ba∤b 一定是形如 a÷b=c⋯da \div b=c\cdots da÷b=c⋯d ，那么相当于去枚举一个 ccc 。我们可以知道 b×(c+1)b\times( c+1)b×(c+1) 就是第一个能整除 bbb 的整数，一开始是 c×bc\times bc×b 现在是 (c+1)×b(c+1)\times b(c+1)×b ，所以多了一个 bbb ，之前比 c×bc\times bc×b 多了 ddd 最终所需要的答案也就是 b−db-db−d。

C Ternary XOR

题目描述

一个三进制数各位必定由 0,1,20,1,20,1,2 组成，例如 1022,11,21,20221022,11,21,20221022,11,21,2022。

给定一个三进制数 xxx,xxx 的首位必定为 222，其余位数有可能是 000 或 111 或 222 。

我们定义三进制的异或操作符 ⊙⊙⊙，对于两个长度均为 nnn 的三进制数 a,ba,ba,b ，a⊙ba⊙ba⊙b 的结果为 ccc，$c_i=(a_i+b_i) \bmod 3 $。这也就是说，将 a,ba,ba,b 相应的数字相加并除以 333。例如，10222⊙11021=2121010222⊙11021=2121010222⊙11021=21210。

你需要找到两个长度为 nnn 且没有前导零的三进制数 a,ba,ba,b 使得 a⊙b=xa⊙b=xa⊙b=x 且 max⁡{a,b}\max \{a,b \}max{a,b} 最小。

输入格式

第一行一个正整数 t(1≤t≤104)t( 1 \le t \le 10^4)t(1≤t≤104)。

之后 ttt 组数据每组数据第一行一个正整数 n(1≤n≤5⋅104)n(1 \le n \le 5 \cdot 10^4)n(1≤n≤5⋅104) 表示 xxx 的长度为 nnn，第二行一个长度为 nnn 的三进制数 xxx （xxx 由 nnn 个 000 或 111 或 222 组成且没有前导零）。

保证所有组数据 nnn 的总和不超过 5⋅104(∑n≤5⋅104)5 \cdot 10^4(\sum n \le 5 \cdot 10^4)5⋅104(∑n≤5⋅104)。

输出格式

输出共 ttt 组，每组两行，两个使得 a⊙b=xa⊙b=xa⊙b=x 且 max⁡{a,b}\max \{a,b \}max{a,b} 最小的长度为 nnn 的三进制数 a,ba,ba,b。

如果有多组 a,ba,ba,b 满足条件，输出任意一种。

###解法

如果要使 max⁡{a,b}\max \{a,b \}max{a,b} 最小，那么一定不存在进位。

对于 a=ba=ba=b

  1. xi=0x_i=0xi​=0 ai=bi=0a_i=b_i=0ai​=bi​=0
  2. xi=1x_i=1xi​=1 ai=1,bi=0a_i=1,b_i=0ai​=1,bi​=0
  3. xi=2x_i=2xi​=2 ai=bi=1a_i=b_i=1ai​=bi​=1

对于 a>ba>ba>b

  1. xi=0x_i=0xi​=0 ai=bi=0a_i=b_i=0ai​=bi​=0
  2. xi=1x_i=1xi​=1 ai=0,bi=1a_i=0,b_i=1ai​=0,bi​=1
  3. xi=2x_i=2xi​=2 ai=0,bi=2a_i=0,b_i=2ai​=0,bi​=2
